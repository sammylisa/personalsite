Around the Web

Ecolab Internship

June 2020 - August 2020

In just under two months, I learned API development, using this skill to develop three MuleSoft APIs. In essence, these APIs grabbed HTTP requests and translated them to SQL queries to retrieve datasets from a database, handling any errors along the way. Tools and languages used include MuleSoft, Anypoint Platform, Anypoint Studio, DataWeave, RAML, and Azure DevOps.

RA Co-op

May 2019 - December 2019

During my time at Rockwell Automation, I was placed in a project to build an integrated development environment (IDE for short). For the first half of the co-op, I worked primarily on the frontend, until I switched teams halfway through and began working between both the frontend and backend as well. During my time at Rockwell, I participated in occasional technical workshops and discussions. Technologies used include TypeScript, HTML, CSS, GraphQL, XText, and Git.

Shared Power Network (Contract)

November 2019 - December 2019

Shared Power Network is an organization that works with local non-profits to provide solar power for the community. As web developer, it was my job to create an interactive page that allows users to see real-time updates of the power output of individual solar panels. The website has not been released yet, as community solar panel farms are still in the process of being built. Technologies used in this project include HTML, CSS, JavaScript, jQuery, and Bootstrap.

Ecolab Hackathon

July 29-30, 2020

My team of four developed a tool using Microsoft Azure that tracks food inspections provided by the city of Minneapolis, targeting businesses that fail their inspections. This tool allows Ecolab to reach out to these businesses proactively before their scheduled follow-up inspection two weeks later. This tool incorporated visualizations, machine learning (to determine why businesses fail), and a notification system.

Our work payed off as we won the hackathon!

Legend of Zelda NES Game Recreation

January 2020 - May 2020

For this project, I worked on a team of 6 to re-create the underworld of the Legend of Zelda game released in 1986 for NES. Together, we developed all 9 dungeons to match the original game, along with some extra enhancements. The game was developed using C# and the Monogame framework, and Git was used for version control.

Windows users can download and play the game here!

NBA Sophomore of the Year

November 2018

For my first "real" personal project, I wanted to include my love of the NBA to keep things fun. So, I set out to find who the best second-year player was. I scraped data from a simple table I found online containing the stats of the relevant players, calculated a score for based on their stats and my scoring metric, and created a simple HTML table to display the results. Besides HTML, Java and jsoup were used in this project.
